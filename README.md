# Customer_behavior_analysis
End-to-end customer shopping analytics solution using Python, SQL, MySQL, and Power BI for data cleaning, business analysis, KPI reporting, and interactive dashboard visualization.
# Customer Shopping Behavior Analytics Dashboard

## Business Problem

Retail businesses need to understand customer shopping behavior, product demand, discount usage, subscription impact, and revenue contribution across customer groups. This project analyzes customer purchase data and converts it into actionable business insights using Python, SQL, and Power BI.

## Project Objective

To clean, analyze, and visualize customer shopping behavior data and build an interactive dashboard for business decision-making.

## Tools Used

* Python
* Pandas
* MySQL
* SQL
* Power BI

## Project Workflow

Dataset → Python Data Cleaning → Feature Engineering → MySQL Database → SQL Analysis → Power BI Dashboard → Business Insights

## Key Features

* Missing value treatment
* Column cleaning and renaming
* Age group creation
* Purchase frequency mapping
* Redundant column removal
* SQL business analysis
* Interactive Power BI dashboard
* Customer revenue and product insights

## SQL Analysis Performed

* Revenue by gender
* Customers using discounts with high purchase amount
* Top-rated products
* Shipping type purchase comparison
* Subscriber vs non-subscriber spending
* Products with highest discount usage
* Customer segmentation based on previous purchases
* Top products by category
* Repeat buyer subscription analysis
* Revenue contribution by age group

## Dashboard Features

* Total customers KPI
* Average purchase amount KPI
* Average review rating KPI
* Subscription status donut chart
* Revenue by category
* Customer count by category
* Revenue by age group
* Customer count by age group
* Slicers for subscription status, gender, category, and shipping type
